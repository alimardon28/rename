Kompyuter

Zo'r Kompyuter (ing . computer — hisoblayman), EHM (Elektron Hisoblash Mashinasi) — oldindan berilgan dastur (programma) boʻyicha ishlaydigan avtomatik qurilma. Elektron hisoblash mashinasi (EHM) bilan bir xildagi atama. Biroq, K. hisoblash ishlarini bajarishdan tashqari uning funksiyasi ancha keng . EHMlarning rivojlanishida K. ning bir necha avlodlarini koʻrsatish mumkin. Bu avlodlar element turlari, konstruktiv-texnologik xususiyatlari, mantiqiy tuzilishi, dastur taʼminoti, texnik tafsilotlari, texnikadan foydalanishning qulaylik darajasi bilan bir-biridan farq qiladi.

Ma'lumotlar

  • Html5 haqida ma'lumot

    HyperText Markup Language yoki HTML veb-brauzerda ko'rsatish uchun mo'ljallangan hujjatlar uchun standart belgilash tilidir. Bunga Cascading Style Sheets (CSS) kabi texnologiyalar va JavaScript kabi skript tillari yordam berishi mumkin. Veb-brauzerlar HTML hujjatlarini veb-serverdan yoki mahalliy xotiradan oladi va hujjatlarni multimedia veb-sahifalarida ko'rsatadi. HTML veb-sahifaning tuzilishini semantik jihatdan tavsiflaydi va dastlab hujjatning ko'rinishi uchun signallarni o'z ichiga oladi. HTML elementlari HTML sahifalarining qurilish bloklari hisoblanadi. HTML konstruktsiyalari bilan tasvirlar va interaktiv shakllar kabi boshqa ob'ektlarni ko'rsatilgan sahifaga kiritish mumkin.

  • Css haqida ma'lumot

    Kaskadli uslublar jadvallari (CSS) HTML kabi belgilash tilida yozilgan hujjat taqdimotini tavsiflash uchun foydalaniladigan uslublar jadvali tilidir.[1] CSS HTML va JavaScript bilan bir qatorda World Wide Web tarmog'ining asosiy texnologiyasidir.[2] CSS taqdimot va tarkibni, jumladan, tartib, ranglar va shriftlarni ajratish imkonini berish uchun mo'ljallangan.[3] Bu ajratish kontentga kirishni yaxshilashi mumkin; taqdimot xususiyatlarini spetsifikatsiya qilishda ko'proq moslashuvchanlik va nazoratni ta'minlash; tegishli CSS-ni alohida .css faylida ko'rsatib, bir nechta veb-sahifalarni formatlashni almashish imkonini beradi, bu tizimli tarkibdagi murakkablik va takrorlanishni kamaytiradi; va faylni baham ko'radigan sahifalar va uni formatlash o'rtasida sahifa yuklash tezligini yaxshilash uchun .css faylini keshda saqlashni yoqing.

  • Bootstrap haqida ma'lumot

    Bootstrap - bu HTML, CSS va JS kutubxonasi bo'lib, u ma'lumot beruvchi veb-sahifalarni ishlab chiqishni soddalashtirishga qaratilgan (veb-ilovalardan farqli o'laroq). Uni veb-loyihaga qo'shishning asosiy maqsadi bu loyihaga Bootstrap-ning rang, o'lcham, shrift va tartib tanlovlarini qo'llashdir. Shunday qilib, asosiy omil mas'ul ishlab chiquvchilar ushbu tanlovlarni o'zlariga yoqadimi yoki yo'qmi. Loyihaga qo'shilgandan so'ng, Bootstrap barcha HTML elementlari uchun asosiy uslub ta'riflarini taqdim etadi. Natijada veb-brauzerlarda nasr, jadvallar va shakl elementlari uchun bir xil ko'rinish hosil bo'ladi. Bundan tashqari, ishlab chiquvchilar o'z tarkiblarining ko'rinishini yanada moslashtirish uchun Bootstrap-da belgilangan CSS sinflaridan foydalanishlari mumkin. Masalan, Bootstrap ochiq va quyuq rangdagi jadvallar, sahifalar sarlavhalari, ko'proq ko'zga tashlanadigan qo'shtirnoqlar va ta'kidlangan matnni ta'minlagan.

  • JavaScript haqida ma'lumot

    JavaScript (/ˈdʒɑːvaˌskrɪpt/),[10] odatda JS deb qisqartiriladi, ECMAScript spetsifikatsiyasiga mos keladigan dasturlash tilidir.[11] JavaScript yuqori darajali, ko'pincha o'z vaqtida tuzilgan va ko'p paradigmadir. U dinamik terish, prototipga asoslangan ob'ektga yo'naltirish va birinchi darajali funktsiyalarga ega. HTML va CSS bilan bir qatorda JavaScript World Wide Webning asosiy texnologiyalaridan biridir.[12] Veb-saytlarning 97% dan ortig'i uni veb-sahifa xatti-harakatlari uchun mijoz tomonida ishlatadi,[13] ko'pincha uchinchi tomon kutubxonalarini o'z ichiga oladi.[14] Barcha yirik veb-brauzerlarda foydalanuvchi qurilmasida kodni bajarish uchun maxsus JavaScript dvigateli mavjud. Ko'p paradigma tili sifatida JavaScript voqealarga asoslangan, funktsional va imperativ dasturlash uslublarini qo'llab-quvvatlaydi. U matn, sanalar, muntazam ifodalar, standart ma'lumotlar tuzilmalari va Hujjat Ob'ekt Modeli (DOM) bilan ishlash uchun amaliy dasturlash interfeyslariga (API) ega.
